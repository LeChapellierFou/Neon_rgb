# Neon_rgb
Version: Beta

Neon mod with rgb colors

This mod allows all players to see the neon of other players

The neon mod only works on cars or motorcycles

Use this commands:

/neon_on   (Enable your neon)

/neon_off   (Disable your neon)

/neon_color  (you can use this color : white, blue, electric_blue, mint_green, lime_green, yellow, gold, orange, red, pink, purple)

/color_help (you can add all colors on chat)

/neon_veh_id (The neon mod will only appear on the current vehicle)

/neon_veh_off (the neon mod will appear on all of the player's vehicles)
